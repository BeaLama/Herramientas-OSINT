# DeHashed

## Descripción

**DeHashed** es una plataforma OSINT especializada en la búsqueda de información procedente de filtraciones de datos (*data breaches*). Permite localizar direcciones de correo, nombres de usuario, dominios, direcciones IP y otros indicadores relacionados con credenciales comprometidas.

---

# ¿Cómo funciona?

DeHashed recopila información de filtraciones públicas y la indexa para facilitar su búsqueda.

Permite realizar consultas por:

- Correo electrónico.
- Nombre de usuario.
- Dominio.
- Dirección IP.
- Nombre.
- Hashes.

---

# Usos en OSINT

- Buscar credenciales filtradas.
- Investigar correos electrónicos.
- Analizar dominios.
- Identificar nombres de usuario.
- Verificar exposiciones en brechas de datos.

---

# Información disponible

Dependiendo de la filtración, puede mostrar:

- Correo electrónico.
- Nombre de usuario.
- Contraseña o hash.
- Dominio.
- Dirección IP.
- Fecha de la filtración.
- Servicio afectado.

---

# Tipos de búsqueda

Buscar un correo:

```text
usuario@empresa.com
```

Buscar un dominio:

```text
empresa.com
```

Buscar un nombre de usuario:

```text
usuario123
```

# Casos de uso

- Auditorías de seguridad.
- Investigación de filtraciones.
- Evaluación de exposición pública.

# Ventajas

- Gran base de datos de filtraciones.
- Múltiples filtros de búsqueda.

---

# Limitaciones

- Requiere cuenta para acceder a todas las funciones.
- Solo muestra datos procedentes de filtraciones conocidas.

---

# Buenas prácticas

- Utilizar únicamente con fines legítimos.
- Verificar la información obtenida.
- Documentar las consultas realizadas.

---

# Resumen

DeHashed es una herramienta muy útil para investigaciones OSINT centradas en filtraciones de datos, permitiendo localizar credenciales y otra información pública relacionada con personas, organizaciones y dominios.

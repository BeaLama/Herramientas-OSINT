# TikTok

## Descripción

TikTok es una plataforma de vídeos cortos que permite compartir contenido multimedia. En investigaciones **OSINT** resulta útil para analizar perfiles públicos, tendencias, ubicaciones y eventos.

---

# ¿Cómo funciona?

Los usuarios publican:

- Vídeos cortos.
- Directos.
- Comentarios.
- Hashtags.

Cada publicación puede incluir:

- Descripción.
- Música.
- Fecha.
- Ubicación.
- Etiquetas.

---

# Usos en OSINT

- Identificar perfiles públicos.
- Analizar vídeos.
- Localizar eventos.
- Investigar empresas.
- Buscar contenido mediante hashtags.
- Verificar información.

---

# Búsquedas

Buscar un usuario:

```text
site:tiktok.com usuario
```

Buscar una empresa:

```text
site:tiktok.com empresa
```

Buscar un hashtag:

```text
#Ciberseguridad
```

---

# Información disponible

- Nombre de usuario.
- Biografía.
- Vídeos públicos.
- Seguidores.
- Seguidos.
- Comentarios.
- Hashtags.

---

# Ventajas

- Gran volumen de contenido actualizado.
- Información sobre eventos recientes.
- Excelente para análisis de tendencias.

---

# Limitaciones

- Muchos perfiles son privados.
- El contenido puede eliminarse rápidamente.
- Algunas publicaciones tienen restricciones geográficas.

---

# Buenas prácticas

- Guardar evidencias de publicaciones relevantes.
- Verificar la autenticidad del contenido.
- Contrastar la información con otras fuentes.

---

# Resumen

TikTok es una herramienta útil para investigaciones OSINT relacionadas con contenido audiovisual, eventos y perfiles públicos, especialmente cuando se complementa con otras fuentes abiertas.

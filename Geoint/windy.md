# Windy

## Descripción

**Windy** es una plataforma de información meteorológica que ofrece datos en tiempo real y previsiones sobre viento, lluvia, temperatura, presión atmosférica, nubosidad y otros fenómenos meteorológicos. Es una herramienta muy utilizada en **GEOINT** y **OSINT** para contextualizar investigaciones relacionadas con el clima.

Gracias a sus mapas interactivos y a la integración de diferentes modelos meteorológicos, permite analizar las condiciones atmosféricas de una ubicación concreta tanto en el presente como en fechas futuras.

---

# ¿Cómo funciona?

Windy recopila datos procedentes de diferentes modelos meteorológicos y estaciones de observación para generar mapas dinámicos.

El usuario puede seleccionar una ubicación y visualizar variables como la velocidad del viento, la temperatura o las precipitaciones, además de consultar previsiones meteorológicas y animaciones temporales.

---

# Funciones principales

Windy permite:

- Consultar previsiones meteorológicas.
- Visualizar mapas de viento.
- Analizar precipitaciones.
- Consultar temperatura.
- Observar la nubosidad.
- Analizar presión atmosférica.
- Consultar imágenes de radar y satélite.
- Obtener información sobre calidad del aire y oleaje.

# Usos en GEOINT

Windy resulta útil para:

- Analizar las condiciones meteorológicas durante un evento.
- Verificar la presencia de lluvia o nieve.
- Estudiar la dirección del viento.
- Contextualizar fotografías y vídeos.
- Planificar investigaciones sobre el terreno.
- Complementar análisis geoespaciales.

---

# Casos de uso

## Verificación de imágenes

Comprobar si las condiciones meteorológicas visibles en una fotografía coinciden con los datos registrados.

---

## Investigación de incidentes

Analizar el tiempo que hacía durante un determinado acontecimiento.

---

## GEOINT

Relacionar información meteorológica con imágenes satelitales y datos geográficos.

---

## Navegación

Consultar el estado del mar, el viento o el tiempo antes de realizar desplazamientos.

# API

Windy ofrece una API que permite integrar datos meteorológicos en aplicaciones y proyectos personalizados.

Algunas funciones avanzadas pueden requerir una suscripción.

# Ventajas

- Información meteorológica muy completa.
- Interfaz intuitiva.
- Mapas interactivos.
- Datos en tiempo real.
- Diferentes modelos meteorológicos.
- Muy útil para investigaciones GEOINT.

---

# Limitaciones

- Algunas funciones avanzadas requieren cuenta.
- Las previsiones pueden variar según el modelo utilizado.
- No todos los datos históricos están disponibles de forma gratuita.

---

# Buenas prácticas

- Comparar varios modelos meteorológicos cuando sea posible.
- Verificar la fecha y hora de los datos consultados.
- Complementar la información con imágenes satelitales.
- Documentar las condiciones meteorológicas durante la investigación.

---

# Caso práctico

**Objetivo:** Verificar las condiciones meteorológicas durante un incidente.

Ubicación:

```text
Madrid, España
```

Información esperada:

- Velocidad del viento.
- Temperatura.
- Nubosidad.
- Precipitaciones.
- Presión atmosférica.

---

# Resumen

Windy es una de las plataformas meteorológicas más completas para investigaciones GEOINT y OSINT. Gracias a sus mapas interactivos y a la información proporcionada por distintos modelos meteorológicos, permite contextualizar eventos, verificar condiciones climáticas y complementar el análisis geoespacial con datos atmosféricos precisos.

# Sentinel Hub

## Descripción

**Sentinel Hub** es una plataforma de observación terrestre que proporciona acceso a imágenes satelitales de alta calidad procedentes de diferentes programas espaciales, como **Copernicus Sentinel**, **Landsat**, **MODIS** y otros satélites comerciales.

Es una herramienta muy utilizada en **GEOINT**, análisis geoespacial e investigaciones OSINT para estudiar cambios en el terreno, detectar actividad humana y analizar imágenes de una ubicación en diferentes fechas.

---

# ¿Cómo funciona?

Sentinel Hub recopila imágenes captadas por distintos satélites y las pone a disposición del usuario mediante un visor web y una API.

La plataforma permite seleccionar una ubicación, elegir el satélite y consultar imágenes tomadas en diferentes momentos, facilitando la comparación temporal y el análisis del terreno.

También incorpora diferentes capas de visualización para resaltar vegetación, masas de agua, zonas urbanas o áreas afectadas por incendios e inundaciones.

---

# Funciones principales

Sentinel Hub permite:

- Visualizar imágenes satelitales.

- Analizar cambios en el terreno.

- Aplicar filtros y capas temáticas.

# Satélites disponibles

Sentinel Hub ofrece acceso a diferentes programas de observación terrestre, entre ellos:

- Sentinel-1.
- Sentinel-2.
- Sentinel-3.
- Landsat.
- MODIS.
- Otros proveedores compatibles.

Cada satélite proporciona distintos tipos de información y resolución.

# Limitaciones

- Algunas funciones requieren suscripción.
- La cobertura puede verse afectada por la nubosidad.
- La resolución depende del satélite utilizado.
- No todas las imágenes están disponibles en tiempo real.

# OpenStreetMap

## Descripción

**OpenStreetMap (OSM)** es un proyecto colaborativo de cartografía de código abierto que permite consultar y editar mapas de todo el mundo. A diferencia de otros servicios comerciales, sus datos son mantenidos por una comunidad de voluntarios y pueden utilizarse libremente bajo una licencia abierta.

En el ámbito **OSINT** y **GEOINT**, OpenStreetMap es una herramienta muy útil para localizar lugares, analizar infraestructuras y obtener información geográfica detallada, especialmente en zonas donde otros mapas pueden tener menos información.

---

# ¿Cómo funciona?

OpenStreetMap almacena información geográfica aportada por miles de colaboradores de todo el mundo.

Los datos incluyen carreteras, edificios, caminos, ríos, hospitales, escuelas, estaciones de transporte, puntos de interés y muchos otros elementos cartográficos.

Toda esta información puede visualizarse desde el navegador o utilizarse mediante aplicaciones y herramientas compatibles.

---

# Funciones principales

OpenStreetMap permite:

- Visualizar mapas.

- Obtener coordenadas.

- Analizar infraestructuras.
- Descargar datos cartográficos.
- Editar y actualizar información.

---

# API

OpenStreetMap dispone de una API que permite acceder a los datos cartográficos y utilizarlos en aplicaciones, scripts y herramientas GIS.

Además, proyectos como **Overpass API** permiten realizar consultas avanzadas sobre los datos de OpenStreetMap.

---

# Ventajas

- Código abierto.
- Cobertura mundial.
- Información muy detallada.
- Actualización constante por la comunidad.
- Compatible con numerosas aplicaciones GIS.
- Descarga gratuita de datos.

---

# Limitaciones

- La calidad de los datos puede variar según la región.
- Algunas zonas están mejor documentadas que otras.
- No ofrece imágenes satelitales propias.
- La información depende de las contribuciones de la comunidad.

---

# Buenas prácticas

- Contrastar la información con otras fuentes cartográficas.
- Verificar la fecha de actualización de los datos cuando sea relevante.
- Utilizar las coordenadas para documentar las investigaciones.
- Complementar el análisis con imágenes satelitales cuando sea necesario.

# Resumen

OpenStreetMap es una de las herramientas cartográficas de código abierto más importantes del ámbito GEOINT y OSINT. Gracias a su amplia base de datos colaborativa y a la posibilidad de consultar y descargar información geográfica detallada, constituye un excelente complemento para el análisis de ubicaciones, infraestructuras y rutas en investigaciones digitales.

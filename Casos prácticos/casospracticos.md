# Casos prácticos

Esta carpeta contiene un caso práctico por cada categoría del manual. Su objetivo es mostrar cómo utilizar una herramienta representativa en un escenario real de investigación OSINT.

Cada caso sigue una estructura sencilla y orientada a la práctica, permitiendo comprender el proceso de trabajo y los resultados que pueden obtenerse con la herramienta seleccionada.

## Estructura

- Objetivo
- Herramienta utilizada
- Escenario
- Procedimiento
- Resultados obtenidos
- Conclusiones
